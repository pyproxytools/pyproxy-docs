By default, all requests through the proxy are logged to files.
The `logs/access.log` file contains all requests accepted by the proxy, and the `logs/block.log` file contains requests blocked by the proxy.

## Change the file path
You can change the path of the log files. Here is the associated documentation depending on your configuration.

* [From an .ini file](../configuration/from_ini_file.md)
* [With command argument](../configuration/command_argument.md)
* [Environment variables](../configuration/env_var.md)

## Disable logging
You can also disable logging of accepted, blocked, or both requests. Here is the associated documentation depending on your configuration.

* [From an .ini file](../configuration/from_ini_file.md)
* [With command argument](../configuration/command_argument.md)
* [Environment variables](../configuration/env_var.md)
