## Latest version
For a direct installation of the latest stable version from [PyPI](https://pypi.org/project/pyproxytools/), use the following command:

```bash
pip install pyproxytools
```

You can now launch pyproxy directly from your command line:

```bash
pyproxy
```

## Other version
To install a specific version, specify it explicitly:

```bash
pip install pyproxy==0.4.5
```
